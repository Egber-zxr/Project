

# 开发环境

    开发工具 pycharm
    python版本 3.8
    数据库 mysql5.7
    数据库管理客户端 HeidiSQL
    
 
# 安装
    1、使用 pycharm 导入本项目
    2、使用 HeidiSQL 执行db文件夹下的 初始化数据库.sql
    3、执行 main.py
    4、打开浏览器访问 http://127.0.0.1:8000
    默认用户名 admin 默认密码：1
    
# 注意
    如果要修改数据库连接地址及参数修改 com/config/db_config.py
